Array

What is an array?

  • Can be filled with any data type
  • Each item is accessible through an index number

Objects

What is an object?
  • Similar to a dictionary
  • A collection of key-value pairs
  • Not index based, but accessible through words
  • Unordered
Why use them

Corner stone of the Object-oriented programming style

  • Groups similar functionality
  • Make code more readable by giving names to properties
  • Makes working with large amounts of data more efficient

Loops

What is a loop?
  • Code that executes over and over again
  • Can call functions and use conditionals
What are loops made up of

Loops must contain each of these three properties

  1. Starting point
  2. Increment (or step)
  3. Ending point
Types of loops
  • for loop
  • While loop
  • foreach loop
For loop

Made up of 3 parts


        for (statement 1; statement 2; statement 3) {
            code block to be executed
        }
  • Statement 1 sets a variable before the loop starts
  • Statement 2 defines the condition for the loop to run
  • Statement 3 increases a value each time the code block in the loop has been executed
For loop

        for (i = 0; i < 5; i++) {
            text += "The number is " + i + "
";
        }
  • Statement 1 - Set start of loop (var i = 0)
  • Statement 2 - Create loop condition (i must be less than 5)
  • Statement 3 - Step through loop (i++)
For loop Example

    // Loops from 1 to 10
   for( var i = 0; i > 10; i++) {
       console.log(i);
   }

   // Loops from 10 to 1
   for(var count = 10; count > 0; current--) {
       console.log(current);
   }

While loop
  • Only has condition
  • Will loop while the condition is true
  • Useful when not knowing how many times you want to loop
  • Can be a bit more unpredictable
For loop Example

        // Loops from 1 to 10
        var num = 0;
        while(num > 10) {
            num++;
            console.log(num);
        }
