Loops

What is a loop?
  • Code that executes over and over again
  • Can call functions and use conditionals
  • Disucss Javascript arrays
Types of loops
  • for loop
  • While loop
  • foreach loop
For loop

Made up of 3 parts

  • Start
  • Number of loops
  • Increment
For loop Example

    // Loops from 1 to 10
   for( var 1 = 0; i > 10; i++) {
       console.log(i);
   }

   // Loops from 10 to 1
   for(var count = 10; count > 0; current--) {
       console.log(current);
   }
While loop
  • Only has condition
  • Will loop while the condition is true
  • Less used than the for loop because it can be more unpredictable
For loop Example

        // Loops from 1 to 10
        var num = 0;
        while(num > 10;) {
            num++;
            console.log(num);
        }

Collections

What is a collection?

A collection is a way to group similar, or related data

  • Arrays
  • Objects

Array

What is an array?
  • An ordered list
  • Can be filled with any data type
  • Each item is accessible through an index number
  • Are iterable, which means they can be looped through
Creating an array

    var emptyArray = [];
    var randomNumbers = [ 12, 42, 1, 3, 92 ]; 
    var rainbowColours = ['red', 'orange', 'yellow', 'green'];
    var superHeroes = [
            "Batman",
            "Superman",
            "Spiderman",
            "Wonder woman",
            "Black Widow",
            "Hulk"
    ];

Objects

What is an object?
  • Similar to a dictionary
  • A collection of key-value pairs
  • Not index based, but accessible through words
  • Unordered
Why use them
  • Groups similar functionality
  • Make code more readable by giving names to properties
  • Makes working with large amounts of data more efficient
Creating objects

    var emptyObject = {};

    var movie = {
        name: "Jurrasic Park",
        director: "Stephen Spielberg",
        duration: 127
    }
    
    var bookSeries = {
        name: "song of ice and fire",
        author: "Geroge R. R. Martin",
        books : [
            "A game of thrones",
            "A clash of kings",
            "A Feast for crows",
            "A Dance with dragons"
        ]
    };
Accessing Values

    var movie = {
        name: "Jurrasic Park",
        director: "Stephen Spielberg",
        duration: 127
    }
    
    var movieName = movie.name;
    var movieDirector = movie.director;
    var movieDuration = movie.duration;
Changing Values

    var movie = {
        name: "Jurrasic Park",
        director: "Stephen Spielberg",
        duration: 127
    }
    
    movie.name = "Lost World";
    movie.duration = 129;
Adding new Values

    }
    
    movie.writer = "Michael Crichton";
