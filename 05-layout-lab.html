Floats & Clears Review

What are floats?

Elements with the float property

  • Remain in the document flow
  • Can be pushed left or right
  • Allows other content to flow around them
What can we do with floats?

Beyond just wrapping text

  • Building full page layouts
  • Build smaller sections of a page (nav, footer etc)
Problem with floats

Once you have ﬂoated something, everything after it in the document ﬂow will ﬂoat up and try and ﬁll whatever gap there is

Problem with floats

Using the CSS clear property we are able to return to the normal document flow

Collapsing Parent

If everything inside an element is ﬂoated, that element will have a height of zero

Unless you clear it

Flexbox

What is flexbox?
  • New CSS3 way of creating layouts
  • Avoids having to use clears/floats
  • Two main concepts to learn
    • Flex container
    • Flex item
Flexbox terminology
Flexbox terminology
Flex Container

                    .container {
                        display: flex;
                    }

When you set the display of an element to 'flex' it becomes a flex container and all immediate children elements become flex items of that container.

Flex direction

A way to change the direction of the main axis of the flex container - Example

  • row (default)
  • column
  • row-reverse
  • column-reverse
Flex wrap

Tells flexbox what to do with elements if the element width has exceeded page width - Example

  • nowrap (reduce the width of elements to fit in the row)
  • wrap (Elements will be pushed to a new row)
  • wrap-reverse (Elements will be pushed to a new row opposite to the cross-axis)
Justify Content

Tells flexbox how to align items on the main axis - Example

  • flex-start
  • flex-end
  • center
  • space-between
  • space-around
Align Items

Tells flexbox how to align items on the cross axis - Example

  • flex-start
  • flex-end
  • center
  • baseline
  • stretch
