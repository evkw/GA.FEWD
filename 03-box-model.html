resetting browser css

What is it for?
  • Each browser has some default CSS that applies to all HTML pages
  • This is true for all older versions of browsers as well
  • Using reset.css or normalize.css removes those defaulted css
  • Lets you work confidently from a blank canvas
reset.css
  • Older of the two popular css resets
  • Basically sets padding and margin to 0 for lots of HTML elements
  • Hasn't changed much since 2008
  • Hasn't needed to change much since 2008
Normalize.css
  • Provides better cross-browser consistency
  • Corrects some bugs that exist in older browsers
  • Builds on-top of reset.css and adds much much more
  • Larger than reset.css

CSS Box model

what is the box model?
  • All HTML elements are considered boxes (even circles)
  • Every box has; margin, border, padding and content
  • Combination of all of these features is the 'box model'
