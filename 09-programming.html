Creating loops with pseudocode

Average marks of a class of 10


            total = 0
            grade_counter = 1

            WHILE grade_counter <= 10
                    INPUT grade
                    total = total + grade
                    grade_counter = grade_counter + 1

            class_average = total / 10
            PRINT class_average

What is javacript?
  • Most popular programming language in the world
  • Object-oriented language
  • Used to create interactive effects within web browsers
  • Can interact with elements in the DOM
  • A "weird, poorly designed language"

Brief history of javascript
  • Created in 10 days
  • Released in 1995
  • Currently version ES2017
What can it do?
  • Valdiate information
  • Update html pages
  • Add interactivity
  • Add Animations
  • Visualize data/art
  • Many many more
