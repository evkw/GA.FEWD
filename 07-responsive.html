Layout types

Fluid layout
  • Also known as liquid layout
  • Uses percentages to adjust to changes in screen
  • Components inside have percentage widths
  • Width change with browser window size
  • Box-sizing: border-box helps a lot
  • Liquid fills the parent container width
Elastic Layout
  • All elements are sized using em or rem
  • Makes everything proportional
  • Usually takes a base font-size and scales everything relative to that size
  • When text size is increased, layout increases

Set the viewport
  • Visible area of a web page
  • Smaller on a phone than a desktop
  • width = device-width sets the width of the page to the screen-width of the device
Media queries
  • Visible area of a web page
  • Smaller on a phone than a desktop
