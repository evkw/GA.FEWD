What does a from add to a website?
  • Form is a section of the page that contains interactive controls
  • Allows a user to send information to a server
  • Can perform validation to ensure data is in a good state before being sent
Challenges of a building a good form?
  • Needs to be easy to understand
  • Needs to be short. The longer the form, the less likely users will complete it.
  • Ensure the data you retrieve from a form is valid and consistent

Form Label
  • Allows users to select the label as well as the control.
  • Targets the Id of a form element.
  • Can be used on all form elements.
Form Button
  • Two types of form buttons
  • Reset - Sets the form back to what it was before users entered anything
  • Submit - Submits form info to a server (via onsubmit function)

Styling a form

Using native elements
  • Using the native form elements can scare off users
  • There are lots of libraries to help style forms
  • Bootstrap
  • Materialize
  • Mixing and matching form elements tends to not look great
Bootstrap

  • CSS Section does not require jQuery

Validating a form

Using a library vs Native validation
  • Native validation can have differing results browser to browser
  • IE does required well but thats about it
  • Third party libraries are made to be consistent across browsers
  • Consistency is king. We really like consistent behaviour
Parsley.js
  • Download from here
  • Javascript library to help simplify form validation
  • Provides feedback on the form for users to correct errors
  • Requires jQuery
