Return Functions

Return Values
  • Sometimes we want a method to return a value
  • Useful for splitting large algorithms into smaller bite sized pieces
Return Values

    function multiply (num1, num2) {    
        return num1 * num2; // return is a reserved word
    } 

    multiply(1, 5);     // Calls the function but value is lost
    var result = multiply(1, 5); // result is set to 5
Return Values
  • When return is hit, the function will exit
  • It is always the last line executed of a return function

    function multiply (num1, num2) {    
        return num1 * num2;
        console.log('value sent!') // Wont execute.
    }

Anonymous functions

Anonymous Functions
  • A function without a name
  • Can be used to execute code as soon as its loaded
Return Values

            // Named function
            function hello() {
                console.log("hello");
            }
            hello();

            // Anonymous function assigned to the hello variable
            var hello = function() {
                console.log("hello");
            }
            hello();
IIFE
  • Immediately-invoked function expression
  • A very important anonymous method
  • Runs as soon as it is defined
  • Variables within the expression can not be accessed from outside
IIFE Uses

            ( function () { 
                var name = "Evan Wallace";
            })();

            console.log(name); // Throws an error
IIFE Uses

        (function () { 
            doPageLoadLogic();
            callServer();
            updateStylesViaJS();
        })();
