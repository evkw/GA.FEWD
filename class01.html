GITHUB REVIEW

What is git?
  • Version control system
  • Used by software engineers all over the world
  • Similar to drop box or Google drive, but for code
  • Built to help multiple people collaborate on the same code base
Git key words
  • Repository
  • Clone
  • Commit
  • Push/Pull
What is github?
  • Platform to help make using GIT easier
  • Comes with a great desktop tool which we will use a lot throughout the course
  • Unlimited free public repositories
  • Hosts files which allows us to share final products with others
