Variables

What are variables?
  • We can tell our program to remember values for us to use later on.
  • The action of saving a value to memory is called assignment
  • The entity we use to store the value is called a variable
variables
  • The action of getting the value from a variable is called accessing the variable
  • We will use all the above techniques to store values into variables, and generate new values using existing variables
variable declaration

    var age;  // declares a variable names age
    age = 21; // Sets the value of the variable to 21

    var age = 21; // Declare and set variable in one line
variable re-assignment
  • Variable values can change after they are declared
  • Changing the value is known as 'reassignment'

    var name = "Jo";  
    name = "Amir"; // The variable value has been changed to Amir
variable conventions
  • Variables start with a lower case letter
  • If they contain multiple words, subsequent words start with an upper case letter

    var numOfStudents = 10;
What can you store in a variable?
  • String - text
  • Numbers
  • Boolean
  • Objects
  • Arrays - Collections

Logical operators

IF ELSE statement will always perform one of the two statements


    if(condition === true) {  
        //Do cool stuff
    } else {
        // Do some other cool stuff
    }
Logical operators

ELSE IF statement are for when there are more then 2 possible code blocks to run


    var topic = "JS";
    if (topic === "JS") {
        console.log("You're learning JavaScript");
    } else if(topic === "JavaScript") {
        console.log("You're still learning JavaScript");
    } else {
        console.log("You're learning something else");
    }
Logical operators

SWITCH statements allows you to change control flow to a large number of possibilities


    switch (new Date().getDay()) {
        case 1:
            day = "Monday";
            break;
        case 2:
            day = "Tuesday";
            break;
        case 3:
            day = "Wednesday";
            break;
        default:
            day = "some other day";
    }
Logical operators

It is possible to test for more than one condition in the same if block


    if (name === "GA"  && password === "YellowPencil"){
        //Allow access to internet
    }

These result in the same way
