Functions

What are functions?
  • A reusable section of code that has a purpose and a name
  • Sometimes called methods or subroutines
  • Make code much more readable
  • Basically when done right, it makes code beautiful
What do they do?

They can perform any code!

  • Calculations
  • Animations
  • Change CSS
  • Communicate with a server
  • Anything!
Defining functions

            // A Function Declaration 
            function sayHello() {    
                console.log( "Hello" ); 
            } 
            
            // A Function Expression
            var sayHi = function() {    
                console.log( "Hi" ); 
            };
How are functions called

Unlike normal code, functions will not run until they are called.

How are functions called

        function sayHello() {    
            console.log( "Hello!" ); 
        } 
        
        sayHello(); // The callsite

Parameters

What are parameters?

The previous example functions are not dynamic

Adding parameters allow us to give some additional information to a function and allows us to create dynamic functionality

Parameters

        function sayHello ( name ) {    
            var greeting = "Hello " + name;    
            console.log( greeting ); 
        } 
        
        sayHello( "Fred" ); 
        sayHello( "Bilbo" ); 
        sayHello(); // ???
How are functions with parameters called?

            function multiply (num1, num2) {    
                console.log( num1 * num2 ); 
            } 
    
            multiply(1);        
            multiply(1, 2, 3); 
            multiply(1, 5);

Return Functions

Return Values
  • Sometimes we want a method to return a value
  • Useful for splitting large algorithms into smaller bite sized pieces
Return Values

        function multiply (num1, num2) {    
            return num1 * num2; // return is a reserved word
        } 
    
        multiply(1, 5);     // Calls the function but value is lost
        var result = multiply(1, 5); // result is set to 5
Return Values
  • When return is hit, the function will exit
  • It is always the last line executed of a return function

        function multiply (num1, num2) {    
            return num1 * num2;
            console.log('value sent!') // Wont execute.
        }
